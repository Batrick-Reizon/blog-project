# Blog Project
**Blog Project** is a full-stack blogging web application built using **React, Node.js, Express, and MongoDB**, integrated with **Firebase Authentication** for secure admin login and user management. It offers a smooth and interactive blogging experience where admins can manage posts, and users can explore, read, and like blogs effortlessly.

# Features
- **Secure Login System** – Firebase authentication ensures only authorized admins can create or delete blogs.
- **Admin Dashboard** – Admins can add, edit, or remove blog posts easily.
- **User Access** – Users can browse and read all blog posts without logging in.
- **Like Functionality** – Readers can engage by liking their favorite blogs.
- **MongoDB Database** – Stores all blog data securely and efficiently.
- **Express + Node Backend** – Handles API requests and connects frontend with the database.
- **Responsive UI** – Built with React for a seamless and modern user experience across all devices.

# Technologies Used
# Frontend
- **React** – for building the dynamic user interface
- **HTML5** – for structure and layout
- **Tailwind CSS** – for modern and responsive styling

# Backend
- **Node.js** – for server-side functionality
- **Express.js** – for handling routes and API endpoints
- **MongoDB** – for database storage and blog management
- **Firebase Authentication** – for secure login and admin control

# How It Works
- **Admin Login** – Admin logs in using Firebase Authentication.
- **Create & Manage Blogs** – Admin can post new blogs or delete existing ones.
- **User Interaction** – Visitors can read blogs and like them.
- **Data Handling** – Blog data is stored and retrieved from MongoDB via Express APIs.
