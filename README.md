# Blog APP (MERN Stack)
**Blog App** is a full-stack web application built using the **MERN Stack (MongoDB, Express.js, React, Node.js)**. It allows users to **create, view, and manage** blog posts with a modern UI and a powerful backend for handling data and requests.

# Features
- **🔐 Firebase Authentication** – Secure login system for user access
- **🧑‍💻 Admin Control** – Only authenticated admin can create and delete blog posts
- **📝 Create Blog Posts** – Users can add new blog content
- **📖 View Blogs** – Display all blog posts dynamically
- **❤️ Like Feature** – Users can like blog posts
- **✏️ Edit Blogs** – Update existing blog content
- **❌ Delete Blogs** – Remove unwanted blog posts
- **💾 MongoDB Storage** – All blog data is stored securely
- **⚡ Real-time Updates** – Changes reflect instantly using React
- **📱 Responsive Design** – Works across all devices

# Technologies Used
# 🎨 Frontend
- **React** – for building dynamic UI
- **Tailwind CSS/CSS3** – for styling and responsive design
- **HTML5** – for structure
- **Axios/Fetch API** – for API communication

# ⚙️ Backend
- **Node.js** – for server-side runtime
- **Express.js** – for API routing and middleware

# 🗄️ Database
**MongoDB** – for storing blog data

# 🔐 Authentication
- **Firebase Authentication** – for secure login and user management

# ⚙️ How It Works
- User creates a blog post from the frontend.
- The data is sent to the backend via API requests.
- Backend processes and stores the data in MongoDB.
- Blogs are fetched and displayed dynamically on the UI.
- Users can edit or delete blogs, and updates reflect instantly.
